# Changelog

## 1.0.0

- Added the initial `GoLand Exact Dark` theme and `GoLand Exact Icons`
- Added Go-specific editor decorations for package names, constants, qualifiers, calls, members, and interface methods
- Added clickable inline implementation hints for interface methods
- Added a Reveal Active File button in the Explorer title bar
- Hardened matching to better ignore comments and strings
- Improved packaging validation and release metadata
- Added Marketplace-ready icon, gallery banner, support documentation, and package ignore rules
